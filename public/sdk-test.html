Implementation Manual

This page demonstrates the full host/client contract. The iframe connects through post-me, receives state events, and invokes host methods to drive the M8 cursor and values.

1) Connect and Fetch Initial State

const m8 = await createM8Client({ debug: false });
const state = await m8.fetchState();
console.log(state.viewName, state.textUnderCursor);

2) Numeric Value APIs

await m8.setValueToHex(0x3f);
await m8.setValueToInt(63);

Host behavior: edit+up/down is coarse movement, edit+left/right is fine movement.

3) Note API

await m8.setNote("C#4");
// valid pattern: [A-G][#-][0-9A-F]
// examples: C-1, D#3, G-B

If a scale restricts notes, movement may stop at the closest reachable note.

4) String/List API

// exact full-text match
await m8.setValueToString("sine", true);

// partial match anywhere in the text
await m8.setValueToString("sin", false);

// search in full current line instead of cursor text only
await m8.setValueToString("filter", false, true);

5) Recommended Event Wiring

m8.onStateChange((state) => renderState(state));
m8.onViewChange((name, title) => log("view", name ?? title));
m8.onTextUpdate((text) => log("text", text));
m8.onKeyPress((keys) => log("keys", keys));
